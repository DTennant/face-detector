## Example
![](https://github.com/DTennant/face-detector/blob/master/data/test.jpg?raw=true)

## Environment
- python 2.7.13 or 3.5.2
- OpenCV (`import cv2`) 3.4.0
- matplotlib 2.1.1
- numpy 1.13.3

## Usage
```
python face_detector.py
```
